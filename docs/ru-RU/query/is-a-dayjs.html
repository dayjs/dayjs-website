Является ли Dayjs объектом

Здесь отображается то, является ли переменная объектом Day.js или нет.

dayjs.isDayjs(dayjs()) // true
dayjs.isDayjs(new Date()) // false

Оператор instanceof также хорошо работает:

dayjs() instanceof dayjs // true
