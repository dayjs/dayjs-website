Objeto

Puede crear un objeto Dayjs especificando algunas de las unidades en un objeto.

Esto requiere el ObjectSupport plugin para trabajar 

dayjs.extend(objectSupport)
dayjs({ hour:15, minute:10 });
dayjs.utc({ y:2010, M:3, d:5, h:15, m:10, s:3, ms: 123});
dayjs({ year :2010, month :3, day :5, hour :15, minute :10, second :3, millisecond :123});
dayjs({ years:2010, months:3, date:5, hours:15, minutes:10, seconds:3, milliseconds:123});

las claves day y date significan ambos día del mes.

dayjs({}) devuelve la hora actual.

Ten en cuenta que como new Date(year, month, date), los meses son con índice 0.
