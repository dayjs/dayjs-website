Day.js

Быстрая 2kB альтернатива Moment.js с тем же современным API

Начало работы

Почему Day.js?

2kB

Меньше JavaScript необходимого для скачивания, парсинга и выполнения, тем самым сохраняя больше времени на написание собственного кода.

Простая

Day.js - это миниатюрная JavaScript библиотека, которая парсит, валидирует, управляет, и отображает даты и время для современных браузеров, обладающая большой совместимостью с Moment.js API.

Если вы используете Moment.js, вы уже знаете как пользоваться Day.js.

Неизменная

Все операции API, которые меняют объект Day.js, вернут новый экземпляр.

Это помогает предотвратить ошибки и избежать длительных отладочных сессий.

I18n

Day.js обладает великолепной поддержкой интернационализации. Но ни одна из локализаций не будет включена в вашу сборку до тех пор, пока вы не начнёте её использовать.
